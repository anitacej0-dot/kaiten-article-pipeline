# Ассеты обложек (фоны)

Сюда положить 4 готовых фирменных фоновых файла (PNG, из проекта обложек):

- `myatnyy.png` — мятный (салатовый/зелёный)
- `zheltyy.png` — жёлтый
- `krasnyy.png` — красный
- `fioletovyy.png` — фиолетовый

Файлы бинарные и в репозиторий пока не добавлены — их нужно скопировать из проекта обложек.
Навык `make-cover` использует ровно эти фоны **без изменений**.
