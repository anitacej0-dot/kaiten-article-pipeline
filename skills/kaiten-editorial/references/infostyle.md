# Инфостиль: правила и примеры «нет/да»

Открывай этот файл при редактуре. Здесь развёрнутые правила с примерами из редполитики Kaiten.

## Оглавление
- Предложения и лишнее
- Оценочные суждения
- Заимствования и жаргон
- Модальные глаголы
- Пассивный залог
- Инверсия
- Журналистские штампы
- «для того чтобы»
- Списки (оформление)
- Вопросы для вовлечения

## Предложения и лишнее

Предложение ≤ 15 слов; меньше — лучше. Убирай:

- усилители без статистики: «самый», «лучший», «наиболее»;
- громоздкие конструкции: «осуществляли поддержку» → «поддерживали»; «компания, оказывающая услуги по удалённому ведению бухгалтерии» → «онлайн-бухгалтерия»;
- плеоназмы: «мобильное приложение для смартфонов» → «приложение для смартфонов»;
- канцелярит: «в целях решения проблемы» → «чтобы решить проблему».

## Оценочные суждения

Вместо оценок — факты, источники, цитаты. Касается и смысла, и отдельных слов.

| Нет | Да |
| --- | --- |
| Всем известно, что в крупных компаниях коммуникации сложнее | Исследование X показало, что в компаниях со штатом более 500 сотрудников… |
| Каждый сталкивался с ситуацией / Эта ситуация знакома многим | Директор по маркетингу Анна А. из ООО «X» рассказала… |
| Самый популярный сервис | Сервис используют 50% IT-компаний России |
| Интересная статья на тему | Статья про делегирование |

## Заимствования и жаргон

Заменяй, если звучит естественно. Прижившиеся и понятные слова оставляй.

| С осторожностью | Используем |
| --- | --- |
| таск, таска | задача, карточка |
| фича | функция, возможность |
| аттач | файл, вложение |
| колл | звонок, встреча, совещание |

Оставляем как есть: баг/ошибка, мессенджер, компьютер (не «ЭВМ»). Не заменяй слова, которые по-русски звучат нелепо.

## Модальные глаголы

Не обозначают действие — превращай в живые глаголы. Исключение: физическая возможность или долг.

| Нет | Да |
| --- | --- |
| провели оптимизацию / осуществили переход | оптимизировали / перешли |
| вам нужно подготовить отчёт / от вас требуется подготовить отчёт | подготовьте отчёт |
| я не открываю документ, потому что нет прав | я не могу открыть документ, потому что нет прав |

## Пассивный залог

Возвращай подлежащее.

| Нет | Да |
| --- | --- |
| В презентации была допущена ошибка | Редактор ошибся в презентации |
| Ключевые задачи не были выполнены в срок | Мы неверно расставили приоритеты и не успели |
| Вам будет отправлен код подтверждения | Мы отправили вам код подтверждения. Проверьте почту |

## Инверсия

Прямой порядок: подлежащее → сказуемое → дополнения.

| Нет | Да |
| --- | --- |
| Задачи в календаре планирует обычно менеджер | Обычно менеджер планирует задачи в календаре |
| В заявках у нас настроены автоматизации | Внутри заявок мы настроили автоматизации |

## Журналистские штампы

Если фраза звучит как из публицистики — не используй, придумай своё.

| Нет | Да |
| --- | --- |
| На встрече были затронуты вопросы | Анна и Алексей обсудили… |
| Как снег на голову | Неожиданно |
| Плачевные последствия | Результат нас не устроил |

## «для того чтобы» → «чтобы»

| Нет | Да |
| --- | --- |
| …добавил столбцы для того, чтобы понять цикл работы | …добавил столбцы, чтобы понять цикл работы |

## Списки (оформление)

**Простое перечисление** — строчные, запятые, последний пункт с точкой:
> Мама купила:
> - апельсины,
> - мандарины,
> - подушку.

**С запятой внутри пунктов** — разделяем «;»:
> - апельсины, которые пахли капустой;
> - мандарины, вызывающие аппетит.

**Пункты-предложения** — с точкой:
> - внимательность. Она всегда знает, что нам нужно;
> - ответственность. Мама держит слово.

**Пошаговые действия** — нумерованный список:
> 1. кипятит воду;
> 2. кладёт ягоды;
> 3. доводит до кипения.

## Вопросы для вовлечения (VK / VC)

Открытые, по теме, без провокаций и конфликта.

| Нет | Да |
| --- | --- |
| Будете ли вы использовать такой сервис? | По каким критериям вы выбираете таск-трекер? |
| Обязан ли сотрудник задерживаться в нерабочее время? | Приходилось ли вам задерживаться, чтобы доделать горящие задачи, и что это дало? |
