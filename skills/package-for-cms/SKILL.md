# Навык: package-for-cms (CMS-пакет)

Финальная упаковка отредактированной статьи к публикации. Этап 8 (см. [CLAUDE.md](../../CLAUDE.md)).

## Вход
- Отредактированная статья (`07_editorial_review.md`).
- База знаний: `knowledge/seo_policy.md` (+ `internal-links-map.md`, когда появится).
- Шаблон: `templates/cms_package.md`.

## Что делать
1. Собери по `seo_policy.md`:
   - **Title** (ключ + год при необходимости), **Description** (ключи + польза);
   - **Post URL** — короткий слаг латиницей по ключу;
   - **теги**; **перелинковка** — ссылки на другие статьи блога с anchor-текстом;
   - **обложка/alt**; **CTA**.
2. Под нейровыдачу Яндекса (research/04): определение в начале на месте, H2–H4, таблицы/списки, **FAQ**, **микроразметка (schema: Article/FAQ)**.
3. Проверки финала: уникальность text.ru ≥90%, нет риска Баден-Бадена (ключи без переспама), факты подтверждены.
4. Заполни `templates/cms_package.md`, сохрани в `08_cms_package.md`. Покажи результат.

## Gate
- Не переводи в публикацию, если: есть неподтверждённые факты, не готов SEO-пакет, нет CTA/перелинковки/обложки, есть риск Баден-Бадена.

## Выход
`08_cms_package.md` — готовый пакет. Публикацию делает человек после финальной проверки.
