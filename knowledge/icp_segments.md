# Сегменты аудитории (ICP)

> Портреты клиентов: кто читает, боли, сценарии, какой язык и примеры уместны. Помогает
> писать под читателя, а не «вообще». Продуктовые факты — в [[products]].
> Источник: скоринг ЦА от 24.02.2026 + ICP overview.

## Сегментация по размеру (лицензии)

- **A+ (Enterprise)** — 250+ лицензий. Якорь выручки, стабильное удержание.
- **B (SMB, средний бизнес)** — 21–250 лицензий. Ключевой двигатель роста.
- **CDX (малый бизнес)** — 1–20 лицензий. Воронка для SMB.

## 8 топ-персон (JTBD: функционально / эмоционально / социально / покупает)

- **🛠 CTO / Руководитель разработки** (IT, финтех). Заменить Jira, не сломав процессы команд / не выбирать между удобством разработки и ИБ / показать совету, что миграция прошла без штрафов. Покупает: self-serve trial → команда → enterprise. Триггер — цена за seat, миграция.
- **📋 PMO Lead** (банк, телеком, ритейл). Видеть портфель 50+ проектов без Excel / отвечать «какой статус?» за 30 секунд / стать PMO, через которого правление решает. Покупает Enterprise с портфельным управлением и аудит-логом.
- **🎨 Управляющий партнёр агентства** (digital, креатив). Вести 30 клиентов без героизма менеджеров / перестать тушить пожары / показать клиентам зрелый процесс. Покупает per-seat + клиентские/гостевые доступы. Триггер — шаблоны, SLA.
- **📊 COO / Операционный директор** (IT, ритейл, услуги). Снять с отделов нагрузку поддержки и фичреквестов / не выгореть диспетчером / метрики. Покупает тариф с Service Desk и BPM. Триггер — учёт времени.
- **🏛 CIO корпорации** (банк, телеком, госкомпания). Импортозамещение Jira/Asana без падения процессов / спать спокойно без санкций / технологический суверенитет управляем. Покупает Enterprise on-prem + SLA + сертификации. Триггер — реестр Минцифры, ГОСТ Р 57580.
- **🛡 Head of Compliance** (финтех, банк, страхование). Полный audit trail без ручного сбора / не бояться проверки ЦБ / защитить лицензию. Покупает аудит-логи, версионность, RBAC в Enterprise.
- **🏗 Директор по производству / ГИП** (производство, стройка, ТОиР). Перевести план и ТОиР с Excel и бумаги в мобильное приложение / не быть виноватым за простой / показать OEE и MTBF. Покупает Sales-тариф с демо, не trial. Триггер — видео-сценарии, кейсы НЛМК/ВкусВилл.
- **👥 HR / Operations Manager** (внутренний пользователь, любой сегмент). Запустить онбординг и базу знаний без IT / перестать отвечать на одни вопросы / «так и должно быть». Покупает стартовый тариф с базой знаний. Триггер — простой UX.

## 5 главных намерений на сайте (intent — затачиваем контент)

1. **«Сравнить с тем, что уже использую»** (Jira, Trello, YouTrack, Excel). Главный intent — сравнительные страницы обязательны.
2. **«Быстро проверить, подойдёт ли нам»** — шаблоны и кейсы по роли/индустрии.
3. **«Стоит ли переезжать с Jira»** — горящий спрос (Atlassian ушёл 12.09.2024, 53% не мигрировали). Нужна страница миграции.
4. **«Увидеть, как это работает на практике»** — не-IT сегменты не идут в trial, нужно видео-демо (45% компаний не доверяют отечественному ПО).
5. **«Понять ограничения и безопасность»** — Финансы и корпораты не покупают без страницы про on-prem, SSO, ИБ.

## Ключевые боли по отраслям (для подбора примеров в статьях)

- **IT (средний бизнес):** зоопарк инструментов (Jira/Trello/Excel), нет единой системы; невозможно гибко адаптировать Jira без программистов; перегрузка поддержки и потеря фичреквестов; сложный онбординг новичков.
- **Крупная IT:** нет единого бэклога идей; нет связи задач с репозиториями кода; тест-кейсы отдельно от разработки → баги в проде.
- **Банк / финтех:** риски блокировки зарубежного ПО и санкций; десятки параллельных проектов без единой картины (PMO); audit trail для ЦБ (Compliance); медленная agile-трансформация в консервативной среде.
- **Телеком:** тысячи обращений без приоритизации, VIP в общей очереди.
- **Ритейл:** координация офис/склады/магазины через разрозненные каналы; срывы запусков промо в пиковые сезоны.
- **Производство:** планы в Excel не синхронны с загрузкой цехов; ТОиР по бумажным журналам; хаос в закупках; ручной сбор контроля качества.
- **Стройка:** координация субподрядчиков через WhatsApp; устаревшая проектная документация.
- **Госсектор:** поручения теряются в почте; разрозненные информационные системы.
