# Целевые ключи по кластерам тем

> Ключи, сгруппированные по темам и интентам ЦА — чтобы статьи не были изолированными, а
> собирались в тематические кластеры (сигнал экспертизы для поиска и нейровыдачи).
> Используется в `make-brief` и `build-outline`. Связано с [[icp_segments]], [[seo_policy]].
>
> ⚠️ Каркас засеян примерами из реальных тем и интентов. **Реальные частотности и расширенные
> списки** ключей команда добавляет из Вордстат/SEO-инструментов.

## Как пользоваться
- Каждый кластер = центральная тема + спутники, связанные перелинковкой ([[internal-links-map]]).
- На каждый ключ — интент одной строкой (что человек хочет).
- Распределять ключи в тексте естественно, **без переспама** (риск Баден-Бадена, см. [[seo_policy]]).

## Кластер 1. Сравнения и аналоги (главный интент ЦА)
Интент: «сравнить с тем, что уже использую» / «найти замену».

| Ключ (пример) | Интент |
| --- | --- |
| аналоги Jira / замена Jira | ушёл Atlassian, ищу российскую замену |
| Kaiten vs Trello / Jira / YouTrack | сравниваю инструменты, нужна таблица |
| аналоги Trello / Asana / Notion | подбираю инструмент под команду |
| аналоги Admin24 (пример реальной статьи) | подборка ITSM/Service Desk |

## Кластер 2. Миграция с Jira
Интент: «стоит ли переезжать и насколько это сложно».

| Ключ (пример) | Интент |
| --- | --- |
| миграция с Jira | планирую переезд, нужен план и риски |
| импорт задач из Jira | технический перенос данных |
| переход с Jira на российский аналог | импортозамещение |

## Кластер 3. Процессы и методологии
Интент: «как выстроить процесс» (часто не-брендовый, экспертный).

| Ключ (пример) | Интент |
| --- | --- |
| как построить service desk (пример реальной статьи) | пошаговый план процесса |
| что такое канбан-метод / scrum-фреймворк | разобраться в подходе |
| управление проектами / задачами в команде | навести порядок в работе |
| ресурсное планирование, WIP-лимиты | настроить конкретный механизм |

## Кластер 4. Отраслевые сценарии
Интент: «подойдёт ли нам» (по сегментам из [[icp_segments]]).

| Сегмент | Темы-ключи (примеры) |
| --- | --- |
| IT | таск-трекер для разработки, замена Jira для команды |
| Производство | управление производством, ТОиР, мобильное приложение для цеха |
| Агентства | управление проектами в агентстве, работа с клиентами |
| Финансы | управление проектами в банке, on-prem, ИБ-требования |

## Кластер 5. Продукт и функции
Интент: «как это работает в Kaiten».

| Тема | Куда ведёт |
| --- | --- |
| Service Desk в Kaiten | kaiten.ru/features/service-desk |
| канбан-доски, диаграмма Ганта, автоматизации | продуктовые страницы |
| тарифы Kaiten (Free/Standard/Pro/Enterprise) | страница цен |

## Что добавить (задача команды)
- Реальные частотности (Вордстат) и приоритеты по кластерам.
- Связки «ключ → существующая/планируемая статья» для перелинковки.
- Минус-слова и сезонность (для торговли/e-commerce — пиковые сезоны).
