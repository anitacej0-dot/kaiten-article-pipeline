# Запрещённые формулировки

> Чёрный список слов и оборотов — основа для HARD/SOFT-правил автопроверки (Vale) и
> AI-предредактуры. Извлечено из [[editorial_policy]]. Формат: ❌ нельзя → ✅ как надо.
> HARD — блокирует сдачу редактору; SOFT — предупреждение.

## Заимствования и жаргон (SOFT — оцениваем по контексту)

| ❌ С осторожностью | ✅ Используем |
| --- | --- |
| таск, таска | задача, карточка |
| фича | функция, возможность |
| аттач | файл, вложение |
| колл | звонок, встреча, совещание |
| ЭВМ | компьютер |
| система обмена сообщениями | мессенджер |

Прижившиеся и понятные слова (баг, ошибка, мессенджер) — допустимы. Не заменяем
заимствование, если русский аналог звучит нелепо.

## Оценочные суждения и неподтверждённые усилители (HARD)

Без фактов и источников нельзя: «самый», «лучший», «наиболее», «самый популярный»,
«всем известно», «каждый сталкивался», «интересная статья».

| ❌ Нет | ✅ Да |
| --- | --- |
| Всем известно, что в крупных компаниях… | Исследование X показало, что в компаниях от 500 сотрудников… |
| Каждый сталкивался с ситуацией | Директор по маркетингу Анна А. рассказала |
| Самый популярный сервис | Сервис используют 50% IT-компаний России |
| Интересная статья на тему | Статья про делегирование |

## Журналистские штампы (HARD)

| ❌ Нет | ✅ Да |
| --- | --- |
| на встрече были затронуты вопросы | Анна и Алексей обсудили |
| как снег на голову | неожиданно |

Плюс рекламные клише: «революционное решение», «инновационный», «уникальный» (без доказательства).

## Нейро-шлак: маркеры нейротекста (HARD)

Обороты, по которым сразу виден AI-текст. Убираем всегда — они фамильярны, водянисты и не в TOV Kaiten.

| ❌ Нет | ✅ Да |
| --- | --- |
| «X живёт в сервисах…», «инструмент живёт рядом с задачами», «рядом живут почта и календарь» | по факту: «работает в связке с…», «встроен в…», «входит в состав…» |
| «общается … как родной» | нейтрально: «интегрируется с…», «связан с…» |
| «единая среда», «единое пространство» (как штамп) | конкретно: какой именно набор инструментов и зачем |
| на старших тарифах | на дорогих тарифах |

Структурный маркер нейротекста в подборках — разнобой подзаголовков карточек. Допустимые пары — только «Преимущества»/«Недостатки», «Возможности»/«Ограничения» или «Плюсы»/«Минусы» (см. [[article_patterns]]).

## Грамматические запреты (SOFT)

- **Модальные глаголы** («вам нужно подготовить», «от вас требуется») → «подготовьте». Исключение — физическая возможность/долг.
- **Пассивный залог** («была допущена ошибка», «вам будет отправлен код») → активный с подлежащим («редактор ошибся», «мы отправили код»).
- **Инверсия** («задачи в календаре планирует обычно менеджер») → прямой порядок («обычно менеджер планирует задачи»).

## Имя бренда (HARD)
