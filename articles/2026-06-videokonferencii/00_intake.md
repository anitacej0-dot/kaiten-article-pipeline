# 00 · Intake — Лучшие сервисы для видеоконференций

| Поле | Значение |
| --- | --- |
| **Тема** | Лучшие сервисы для видеоконференций (подборка) |
| **Тип материала** | SEO-подборка (листикл), 10–15 сервисов |
| **Площадка** | Habr |
| **Дедлайн** | не указан в ТЗ |
| **Источник ТЗ** | SEO-специалист (файл «Лучшие сервисы для видеоконференций — habr.com») |

## Цель
Собрать подборку ВКС-сервисов с описаниями, сравнительной таблицей и блоком «как выбрать».
Нативно представить Kaiten в подборке.

## ЦА
Бизнес, выбирающий сервис видеоконференций: руководители, IT, операционные команды.
Пересечение с нашими сегментами ([[icp_segments]]): IT, агентства, корпораты, госсектор
(импортозамещение Zoom/Webex).

## SEO-вводные (из ТЗ)
- **Ключи:** приложения для видеоконференций; программа для конференций по интернету; программы для видеосвязи; видеоконференция онлайн; платформа для видеоконференций; сервисы видеоконференций.
- Первое предложение под H1 — может содержать побуждение к действию (для сниппета).
- Ключи распределить равномерно; по возможности — в подзаголовках.
- **Структура:** вводный абзац → что такое ВКС → 10–15 сервисов (описание/преимущества/недостатки/цена) → сравнительная таблица → как выбрать ВКС.
- Объём описания одного сервиса: 1200–1500 знаков.
- **Конкуренты в выдаче:** kp.ru, smmplanner, timeweb, okocrm, shtab.app, workspace.vk.ru.
- **Сервисы (15):** Kaiten, Яндекс Телемост, SaluteJazz, Контур Толк, TrueConf, Битрикс24, Compass, VK Звонки, МТС Линк, VideoMost, eXpress, Pruffme, Commfort, IVA, Webex.

## ⚠️ Риски — на решение редактора/SEO до старта
1. **Kaiten — НЕ сервис видеоконференций.** По [[products]] это система управления рабочими процессами; ВКС-модуля нет (Chat/Messenger в продуктовой матрице — 3/10). Включение Kaiten в подборку чистых ВКС — растяжка позиционирования. **Гейт «продуктовая точность» сработает на этапе AI-ревью.**
   - Варианты: (а) подавать Kaiten честно — как среду, где договорённости со встреч превращаются в задачи (не ВКС, а слой управления работой вокруг созвонов); (б) согласовать с SEO замену/уточнение роли Kaiten в подборке.
2. **Площадка Habr** — формально вне узкого MVP (MVP = блог Kaiten). Как пилот-прогон формата «подборка» подходит, но публикацию площадки подтвердить.
3. **Не выдумывать факты** о 14 конкурентах (цены, плюсы/минусы) — research помечает к проверке по официальным источникам на дату.

## Gate (этап 0 → 1)
Все обязательные поля заполнены (цель, ЦА, площадка, тип, SEO-вводные) → можно собирать бриф.
Риск №1 несём дальше как открытый вопрос к редактору.
